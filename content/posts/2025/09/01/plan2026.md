图中的红色竖线是减半的日期，蓝色折线显示了从减半到牛市顶部再到熊市底部，最后到下一次减半的过程。可以发现熊市底部处于牛市顶部和下一次减半中间的某个位置，经过计算，三次分别是 43%、41.4%、42.2%，方差相当小。下面是其中一次的计算过程：
```python
>>> import datetime
>>> datetime.datetime(2024,4,20) - datetime.datetime(2021,11,10)
datetime.timedelta(days=892)
>>> datetime.datetime(2022,11,21) - datetime.datetime(2021,11,10)
datetime.timedelta(days=376)
>>> 376 / 892
0.42152466367713004
```
历史上很多依靠价格（不管是绝对价格还是百分比）来预测熊市底部的尝试最后都失败了，比如下面这些论断：
- 熊市底部价格一定比牛市顶部价格跌去 80% 以上（三次分别跌了86.8%、84.1%、77.6%）
- 熊市底部价格一定不会跌破上轮牛市顶部价格（2022-11-21的 15479 跌破了2017-12-17的 19666）
- 熊市底部价格一定不会跌破矿工关机价格（曾多次因价格大跌导致矿工关机，随后开启反弹或者牛市）

尽管如此，价格依然是一个很好的参考，当出现了某些极端的价格时，哪怕不是底部，通常也距离底部不远了。

我们希望熊市底部出现在定投的过程中，所以定投时间不能太短（要防止因为一些偏差导致熊市底部落在定投时间以外）；同时我们也希望尽可能降低定投成本价，所以定投时间不能太长（时间越长距离熊市底部越远的位置成本就越高）。经过综合考虑，我将这轮定投时间设置为100天。

## 计划明细
本次计划主要按照时间来制定，以牛市顶部到下一次减半的 42% 位置为中心，进行为期 100 天的定投。

按照这个计划，第三次减半和第四次减半期间的定投时间范围是2022年10月1日～2023年1月8日，使用[BTC定投计算器](https://cryptotoolhub.com/zh/tools/crypto-dca-calculator)可以算出定投成本价是17709美元。其他熊市的定投成本可以自行计算，这里不再赘述。

根据[这个网站](https://www.datawallet.com/zh/比特币减半倒计时)的数据，比特币预计在2028年3月30日发生减半。假设这轮牛市的最高点是2025年10月6日的126200美元，我们简单计算一下：
```python
>>> import datetime
>>> datetime.datetime(2028, 3, 30) - datetime.datetime(2025, 10, 6)
datetime.timedelta(days=906)
>>> 906 * 0.42
380.52
>>> datetime.datetime(2025, 10, 6) + datetime.timedelta(days=381)
datetime.datetime(2026, 10, 22, 0, 0)
>>> datetime.datetime(2026, 10, 22) - datetime.timedelta(days=50)
datetime.datetime(2026, 9, 2, 0, 0)
>>> datetime.datetime(2026, 10, 22) + datetime.timedelta(days=49)
datetime.datetime(2026, 12, 10, 0, 0)
```
计算出来的定投时间范围是：**2026年9月2日～2026年12月10日**。

## 手动还是自动
养成习惯有利于长期坚持。对于手动定投来说，选择一个合适的时间，可以是吃饭的时候、午休的时候，或者晚上睡觉前，在这个时间把每天要投的额度从欧易的“赚币”或者币安的“理财”赎回，然后购买比特币。

为了节约交易手续费，对于资产在 \$100000 以内的用户来说，可以使用币安，市价单和限价单优惠后的手续费都是 0.075%（需要提前准备好用来抵扣手续费的 BNB）；对于资产超过 \$100000 的用户，可以在欧易使用限价单进行交易，手续费是 0.045%。

如果你是个懒人，可以在~~欧易的“交易”>“策略交易”>“定投策略”或者~~[^1]币安的“闪兑定投”中设置自动定投。全部要投的资金除以一百得到每天要投的资金，填在从 USDT 一栏；频率设置为每日的任意一个时间；资产目标地址选择“理财-活期资产”；最长期限设置为 100。
[^1]: 欧易的定投必须先把全部要投的本金划转到交易账户，还没买比特币的资金没法吃利息。

和手动定投相比，自动定投的好处是设定好以后几乎不用去管了，也可以避免看到账户浮亏的焦虑。手动定投的好处是更加灵活，假如你发现了更好的理财机会，可以把一部分还没投入计划的资金转过去吃更高的利息。

怎么选择看个人，我更倾向于软件设置自动定投的方式。其实你不用花那么多时间研究行情，大道至简，很多时候你花的时间越多反而越赚不到钱。把你的时间花在能提升自己的事情上，可以学习，可以打游戏，可以出去旅游，人生不止赚钱。

熊市播下种子，牛市收获果实。希望大家都能有一个光明的未来。
